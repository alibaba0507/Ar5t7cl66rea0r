    Article Creator is design to help content creator , blog owners and others to easy create content.The design is very simple and user friendly, but yet powerful and packed with a lot of functionality. Main functionality is based on user keywords to capture high rank articles from web and spin it. Spin of articles is based on algorithm with more 65 000 synonyms also the user have a option to check online and select most suitable words to replace with. This system is design for automation of article creation and speeding the process , but also leave a option to the user to fine tweak end result of article creation.

 

Creating New Project

This is the main screen of the system . Based on user keywords and niche user can create a new project

The project window has three tabs , Project where user can create profile of the current project,

Like Project name , keywords ,and project directory, in the keyword input user can copy keywords from Excel file or text file and paste in it.

User must choose project directory where all the extracted articles will be saved

 To save the project user press “Save” button

Run Selected Project

In the Projects Tab , user can select the saved project , with Right click popup menu is shown:

  • Delete - User will be prompted to delete the project only or option to delete directories where the project has been saved.
  • Run Project will scan search engines and will extract Articles based on user keywords selection

And the result will be displayed on articles tab where the article title and url will be linked to selected keyword.

Setup Search Engine - By default system provide three main search engines and will alternate around , but If user want there custom one , user must click on Options =>  Preferences and popup option dialog will appear the user must select “Search URL” is the search engine url and

“Links syntax” , this is regex syntax to extract links based on search engine tag structure

Fetch Articles

  • Fetch Articles… This function will extract(Download) the articles from the url to the selected Project Directory on the Project Tab.Will count the article words and populate Word Count column , By default the system has built In Article parser , to remove all the scripts , links , buttons , inputs and so on.,but also can be customized by user

       

After the articles have been extracted and words Counted ,user can do single or multiple selection and right click will show popup menu :

  •  Clean Article - This is more refine filter to try to remove more unnecessary tags
  • Spin Articles - will spin all of the selected articles.
  • Email Articles - User can email selected Spun Articles to blog email(blogger , wordpress , e.t.c) if set up. To set up emails user must go to Menu, Options => Preferences ,Email tab

  • GMail address is the User Gmail from where ArticleCreator will send the emails with the articles
  • Gmail password - user gmail login password , it is encrypted so … no snooping
  • Email for Blogger Blog - email that must be set up for selected blog on blogger , wordpress or other blogger campaign. So if you not sure just google how to setup blog publish email …

If “Email Articles” is selected from popup menu will be prompted to choose to witch blog email to be send

Edit Articles and Spinning

If user desire to refine the spun articles , user may double click on selected Article from “Articles” tab and will go to “Edit” tab:

All the test with relevant images will be preserved , except links and buttons and any actions will be removed. The user can edit the article and save it again or Spin Again

On the spin side user can check the changes , to see what change one easy can flip between “Original” and “Spin” tab and will see what change , If user select text and press “Check Selected Text OnLine” the system will find relevant synonyms and will be displayed in console window (bottom text window)
